AWS

The Things Stack is currently available as an AWS CloudFormation stack that can be purchased at the AWS Marketplace, and as a set of CloudFormation templates that can be deployed to build a highly available cluster for large-scale deployments.

For getting started, we recommend AWS Marketplace AMI deployment method.
