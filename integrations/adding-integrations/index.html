Adding Integrations

Integrations allow you to process and act on data by triggering events. Events can be pretty much anything, and can easily be set up using Webhooks or MQTT.

Integrations are managed within applications. Applications can be added to The Things Stack by following instructions for Adding Applications.

Once an application has been added, learn how to work with the Application Server MQTT server and HTTP webhooks.
